# DIY LED Lampe v2
Selbst gebaute Countdown-Lampe auf Basis eines [ESP32](https://de.wikipedia.org/wiki/ESP32)-Mikrocontrollers mit 48 WS2812B-LEDs. Steuerung per Bluetooth LE und WLAN.

📖 **Dokumentation:** [wiki.cogneon.de/DIY_LED-Lampe](https://wiki.cogneon.de/DIY_LED-Lampe)

## Dateien

| Datei | Beschreibung |
|---|---|
| `led-lampe-v2.ino` | Arduino-Sketch (ESP32) |
| `index.html` | Web-Bluetooth-Controller für Chrome/Edge |
